- This demo is used to control the ir cut 
## Install dependence lib
- Install libgpiod
```shell
sudo apt-get install libgpiod-dev
```
## How to build
```shell
make
```
## Execute
- Turn on the ir cut 
```shell
./irCut 1
```
- Turn off the ir cut
```shell
./irCut 0
```
## Clean
```shell
make clean
```
